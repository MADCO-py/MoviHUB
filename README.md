Descripción:
MovieHub es una aplicación web tipo blog enfocada en publicaciones de películas y series. Permite consultar publicaciones, ver detalles, buscar, filtrar y gestionar contenido desde la interfaz.

El proyecto fue desarrollado utilizando HTML, CSS y JavaScript puro, consumiendo una API pública.

Funcionalidades:

Listado de publicaciones con paginación
Vista de detalle de cada publicación
Búsqueda de publicaciones por texto
Filtros por contenido y usuario
Creación de publicaciones con validaciones
Edición y eliminación de publicaciones (simulado)
Sistema de likes (simulado)
Estados de la interfaz: loading, error, success
Navegación mediante sidebar
Diseño responsive

Tecnologías utilizadas:
HTML
CSS
JavaScript
DummyJSON API

API utilizada:
https://dummyjson.com/docs/posts

Endpoints utilizados:
GET /posts
GET /posts/search?q=
POST /posts/add
